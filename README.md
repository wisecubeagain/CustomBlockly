# Blockly Demo

这是一个简单的 Blockly 测试 Demo，可以体验图形化编程。

## 快速开始

### 1. 安装依赖（如果还没安装）

```bash
npm install
```

### 2. 启动开发服务器

```bash
npm start
```

或者

```bash
npm run dev
```

### 3. 打开浏览器

Vite 开发服务器会自动在浏览器中打开 `http://localhost:8080`，或者手动访问该地址。

**注意**：本项目使用 Vite 作为开发服务器，支持 ES 模块导入，解决了浏览器直接运行时的模块解析问题。

## 功能说明

- **左侧**：Blockly 图形化编程工作区，可以拖拽积木进行编程
- **右侧**：实时显示生成的 JavaScript 代码
- **顶部按钮**：
  - "生成代码"：手动触发代码生成
  - "清空"：清空工作区中的所有积木

## 积木类别

- **逻辑**：条件判断、逻辑运算
- **循环**：重复执行、遍历
- **数学**：数字、算术运算
- **文本**：字符串操作
- **变量**：创建和使用变量
- **函数**：定义和调用函数

## 使用提示

1. 从左侧工具箱拖拽积木到工作区
2. 积木会自动连接（如果类型匹配）
3. 右侧会实时显示生成的 JavaScript 代码
4. 可以点击右上角的缩放按钮调整工作区大小
